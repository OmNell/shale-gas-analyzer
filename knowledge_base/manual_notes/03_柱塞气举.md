# 柱塞气举知识卡片

## 工艺定位

柱塞气举是一种利用井筒内柱塞周期性举升液体的排水采气方式，常用于低产气井、间歇性积液井或泡沫排水效果不稳定的气井。其核心思想是通过柱塞减少气体滑脱和液体回落，提高周期性排液效率。

## 适用条件

- 低产低压气井存在较明显积液问题；
- 泡沫排水采气效果不稳定或持续性不足；
- 井筒条件允许安装柱塞装置；
- 井口压力、套压、油压能够满足柱塞运行周期；
- 有较清晰的开关井制度或可进行周期优化。

## 常见数据特征

- 日产气量长期低位运行；
- 油压波动明显，产液不连续；
- 关井恢复压力后，开井短期产气和产水有恢复；
- 泡排后短期有效，但产量恢复不持久；
- 气水比偏低，井筒携液能力不足。

## 优点

- 相比连续机械举升，能耗较低；
- 可减少液体滑脱，提高排液效率；
- 适合部分低压低产气井长期制度化运行；
- 便于与自动化控制系统结合，基于压力、流量、柱塞到达时间进行优化。

## 风险与限制

- 对井筒通畅性、管柱结构和井口装置有要求；
- 初期制度优化较复杂，需要调整开井、关井和柱塞运行周期；
- 地层能量严重不足时，柱塞可能无法稳定运行；
- 砂、蜡、结垢等问题可能影响柱塞运行。

## Agent 输出建议模板

> 若泡沫排水采气后产气恢复不明显，且油压波动、气水比偏低、间歇生产特征明显，可将柱塞气举作为第二优先级措施。实施前应复核管柱条件、井筒通畅性、压力恢复能力和历史作业记录。
