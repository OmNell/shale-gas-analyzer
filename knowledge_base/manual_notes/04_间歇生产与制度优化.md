# 间歇生产与制度优化知识卡片

## 工艺定位

间歇生产是低产低压气井常见的生产制度优化手段。其基本思想是通过关井恢复压力、开井集中排液和产气，缓解连续低速生产造成的液体累积问题。

## 适用条件

- 连续生产时产气量持续下降；
- 井底积液风险存在，但套压仍有恢复能力；
- 开井初期产量较高，随后快速衰减；
- 关井后压力能够恢复，说明地层仍有一定供气能力；
- 需要低成本验证积液与能量不足问题。

## 典型数据特征

- 开井后产气短期恢复，随后快速下降；
- 关井恢复压力后再次开井出现产水峰值；
- 油压和套压随开关井呈周期性变化；
- 连续生产时油压波动更明显，间歇生产后波动减弱。

## 与泡排、柱塞的关系

- 间歇生产可与泡沫排水配合，通过关井恢复压力、开井强排带出积液。
- 若间歇生产有效但持续性不足，可评估柱塞气举。
- 若间歇生产后压力恢复很弱，可能说明地层能量衰竭程度较高，需谨慎投入高成本措施。

## 风险与限制

- 过长关井可能影响累计产量；
- 过短关井不能有效恢复压力；
- 需要结合经济产量、管网压力和现场管理制度；
- 不能替代对压力传感器和作业记录的复核。

## Agent 输出建议模板

> 当前井若表现为低产、油压波动、气水比下降但套压仍可恢复，可建议进行“间歇生产 + 泡排”的低成本组合试验。建议记录开关井周期、开井初期产气、产水峰值、压力恢复幅度，并据此判断是否升级为柱塞气举或其他排水采气措施。
