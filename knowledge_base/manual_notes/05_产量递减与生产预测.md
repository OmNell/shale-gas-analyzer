# 产量递减与生产预测知识卡片

## 知识定位

产量递减分析是气井生产动态分析和产能预测的重要基础。对于页岩气井，生产通常表现出初期高产、快速递减、随后进入较缓慢衰减阶段的特征。递减规律可用于识别异常生产、预测未来产量、估算最终可采量和评价措施效果。

## 常用方法

- 传统递减曲线分析：指数递减、双曲递减、调和递减等。
- 数据驱动预测：LSTM、GRU、XGBoost、TimesNet 等时序模型。
- 混合方法：结合地质工程特征、物理约束和机器学习模型。
- 迁移学习：将已有区块或相邻区块知识迁移到样本较少的新区块。
- 不确定性预测：通过预测区间或 conformal prediction 表达预测可信度。

## Agent 诊断中的作用

递减曲线不仅用于预测未来产量，也可用于判断当前变化是否异常：

- 若产量下降符合长期递减趋势，可能偏向自然衰竭。
- 若短期下降幅度远超长期递减趋势，应复核井底积液、制度调整、设备异常或数据质量问题。
- 若措施后产量短期恢复但很快再次下降，说明措施效果可能不稳定。
- 若压力与产气同步下降且产水不高，可能偏向地层能量衰竭或裂缝导流能力下降。

## 与 RAG 的结合方式

RAG 可以检索：

- Arps/DCA 模型适用条件；
- 页岩气快速递减机理；
- 生产预测模型对比；
- 迁移学习在页岩气预测中的应用；
- 物理约束机器学习在油气生产预测中的应用。

## Agent 输出建议模板

> 当前井最近 30 天产量下降速度明显高于长期递减趋势，不能简单归因于自然递减。建议结合压力、产水、气水比和作业记录复核井底积液、生产制度变化或数据异常；若确认无作业干扰，可进一步采用递减曲线与数据驱动模型评估未来产能。
