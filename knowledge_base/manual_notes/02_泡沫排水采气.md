# 泡沫排水采气知识卡片

## 工艺定位

泡沫排水采气是一类低成本排水采气措施，常用于低产低压气井或存在井底积液风险的气井。其基本思想是通过起泡剂降低液体表面张力，使井筒液体形成泡沫体系，降低液体有效密度和携液难度，从而提高气流带液能力。

## 适用条件

- 气井处于低产低压阶段，但尚有一定套压或地层供气能力。
- 日产气量下降明显，气水比降低，疑似携液能力不足。
- 油压波动、油套压差异常，可能存在井筒液柱扰动。
- 井筒和地面流程具备加注起泡剂条件。
- 作为早期、低成本、可快速验证的干预措施。

## 典型触发信号

- 当前日产气量低于经济稳定生产水平；
- 产气短期内下降过快，超过正常递减；
- 气水比降低；
- 产水不高但产气持续下降；
- 套压仍有恢复能力，不属于完全能量衰竭。

## 优点

- 成本相对较低；
- 施工简单，适合快速试验；
- 对低产低压井有较强现场适用性；
- 可作为柱塞气举、速度管柱等措施前的优先验证方案。

## 风险与限制

- 起泡剂类型需要结合矿化度、凝析油、温度、井筒条件选择。
- 高含水或严重积液井可能需要更强排液措施。
- 如果地层能量已经严重不足，泡排效果可能有限。
- 如果压力数据异常，必须先复核仪表和作业记录，避免误判。

## Agent 输出建议模板

当数据表现为“产气快速下降 + 气水比降低 + 油压波动 + 套压尚有恢复能力”时，可建议：

> 当前井表现出携液能力不足或井底积液风险，建议优先进行数据复核，并考虑低成本泡沫排水采气试验。实施后应在 24～72 小时内复核日产气、日产水、油压波动和气水比变化，若效果不稳定，再评估柱塞气举或制度优化。
